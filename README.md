Requirements
------------------------
* JDK 17 <br/> (tested with Amazon Coretto 17.0.13)

  ```
  --add-exports java.base/sun.nio.ch=ALL-UNNAMED
  ```
* SBT <br/> (tested with Version 1.10.5)

Description
------------------------
